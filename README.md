# ピアノコンクールノート PWA

無料で長期運用するための静的PWA版です。

## 特徴

- サーバー不要
- DB不要
- 外部Storage不要
- データは端末内のlocalStorageに保存
- 録音はブラウザのMediaRecorderを使用
- JSONバックアップ/復元に対応

## 使い方

GitHub Pagesで公開されたURLをスマホで開き、ホーム画面に追加します。

公開URL:

https://kamihikooki424-ai.github.io/piano-competition-note-pwa/

## 注意

データは端末内に保存されます。機種変更や端末故障に備えて、設定画面からバックアップを書き出してください。

## 公開範囲

このリポジトリはpublicです。HTML/CSS/JavaScriptなどのアプリ本体コードは一般に見えます。

ただし、利用者が入力したコンクール情報、練習回数、録音データはGitHubへ送信されません。データは利用者の端末内に保存されます。
