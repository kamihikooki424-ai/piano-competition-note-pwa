# ピアノコンクールノート PWA

無料で長期運用するための静的PWA版です。

## 特徴

- サーバー不要
- DB不要
- 外部Storage不要
- データは端末内のlocalStorageに保存
- 録音はブラウザのMediaRecorderを使用
- 先生の練習指示を回数で管理
- 6項目採点から総合点を自動計算
- 点数推移を表示
- 先生に渡す共有メモを作成
- 初回案内で使い始めの手順を表示
- 録音名と採点履歴の編集に対応
- 試用者向け案内文とフィードバック記録に対応
- 端末内データ診断とバックアップ忘れ防止表示に対応
- 録音ファイルの端末保存確認に対応
- サンプルデータから自分用データへ切り替えるセットアップ導線に対応
- 先生チェック依頼文の自動作成とコピーに対応
- 日々の練習時間・練習内容の記録に対応
- 直近7日の練習サマリーと連続練習日数の表示に対応
- 締切・練習・録音・採点から本番準備度と次アクションを表示
- 子どもプロフィールを登録し、ホーム・先生共有メモ・バックアップに反映
- コンクールごとの申込手続きチェックリストに対応
- 課題曲・作曲者・目標テンポ・注意点の管理に対応
- 先生コメントと次回課題の記録に対応
- ホームの最新状況カードで曲・練習・先生コメント・録音を確認
- JSONバックアップ/復元に対応

## 使い方

GitHub Pagesで公開されたURLをスマホで開き、ホーム画面に追加します。

公開URL:

https://kamihikooki424-ai.github.io/piano-competition-note-pwa/

## 注意

データは端末内に保存されます。機種変更や端末故障に備えて、設定画面からバックアップを書き出してください。

## 公開範囲

このリポジトリはpublicです。HTML/CSS/JavaScriptなどのアプリ本体コードは一般に見えます。

ただし、利用者が入力したコンクール情報、練習回数、録音データはGitHubへ送信されません。データは利用者の端末内に保存されます。
