# ピアノノート PWA

無料で長期運用するための静的PWA版です。

## 特徴

- サーバー不要
- DB不要
- 外部Storage不要
- データは端末内のlocalStorageに保存
- 録音はブラウザのMediaRecorderを使用
- 先生の練習指示を回数で管理
- 6項目採点から総合点を自動計算
- 点数推移を表示
- 先生に渡す共有メモを作成
- 初回案内で使い始めの手順を表示
- ホームから公開URLをコピーして試用者へ共有可能
- ホームと練習画面を子ども向けの最小表示に整理
- 子どもが見る見出し・ナビを短い日本語に整理
- スタンプが日付マスへ大きく貼られるアニメーションに対応
- 子どもホームを「今日やること」「7日スタンプ」中心の表示に整理
- 「できた」時に大きな達成演出を表示
- 録音画面で音声録音と動画録画を切り替えて端末内保存可能
- 動画録画前に容量が大きいことを短く案内
- 子ども向け画面の文字量と縦の長さを削減し、主ボタンを強調
- ホームの「今日のピアノ」背景画像を新しいピアノイラストへ差し替え
- レッスン予定の登録・一覧表示・次回予定表示に対応
- アプリ名を「ピアノノート」に変更
- 下タブを文字だけのシンプルな表示に変更
- ホームに1ヶ月分のレッスン・準備・締切・本番予定を表示
- 予定に種類を設定し、準備や申込なども登録可能
- ホームから主要画面へ移動できるボタンを表示
- 録音名と採点履歴の編集に対応
- 試用者向け案内文とフィードバック記録に対応
- 試用件数・平均評価・次に見ることを試用サマリーで確認可能
- 複数人に試してもらうための試用チェックリストに対応
- 端末内データ診断とバックアップ忘れ防止表示に対応
- 録音ファイルの端末保存確認に対応
- サンプルデータから自分用データへ切り替えるセットアップ導線に対応
- 先生チェック依頼文の自動作成とコピーに対応
- 先生コメントから家庭練習用の課題カウンターを作成可能
- 先生コメント保存時に次回課題を練習カウンターへ同時追加可能
- ホームの今日のミッションから練習課題を1タップで進められる
- ホームの「できた」ボタンで今日のシールを獲得できる
- ホームで直近7日分のスタンプカードを確認できる
- 練習画面のこども練習モードで、今日の課題・できた・タイマー・録音を大きく表示
- こども練習モードの「できた」から簡易練習記録を自動作成
- 日々の練習時間・練習内容の記録に対応
- 今日の練習記録に応じたシール表示で継続を促す
- 直近7日の練習サマリーと連続練習日数の表示に対応
- 締切・練習・録音・採点から本番準備度と次アクションを表示
- 子どもプロフィールを登録し、ホーム・先生共有メモ・バックアップに反映
- コンクールごとの申込手続きチェックリストに対応
- 課題曲・作曲者・目標テンポ・注意点の管理に対応
- 先生コメントと次回課題の記録に対応
- ホームの最新状況カードで曲・練習・先生コメント・録音を確認
- JSONバックアップ/復元に対応

## 使い方

GitHub Pagesで公開されたURLをスマホで開き、ホーム画面に追加します。

公開URL:

https://kamihikooki424-ai.github.io/piano-competition-note-pwa/

## 注意

データは端末内に保存されます。機種変更や端末故障に備えて、設定画面からバックアップを書き出してください。

## 公開範囲

このリポジトリはpublicです。HTML/CSS/JavaScriptなどのアプリ本体コードは一般に見えます。

ただし、利用者が入力したコンクール情報、練習回数、録音データはGitHubへ送信されません。データは利用者の端末内に保存されます。
